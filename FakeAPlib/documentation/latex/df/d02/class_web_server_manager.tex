\doxysubsection{Descripción detallada}
Clase que gestiona la configuración y operación de un servidor web y DNS, junto con el manejo de archivos por solicitudes HTTP. 

Esta clase ofrece funcionalidades para\+:
\begin{DoxyItemize}
\item Iniciar un portal cautivo mediante DNS y redirigir peticiones.
\item Registrar rutas (endpoints) personalizados en el servidor web (p.\+ej. /admin-\/panel, /submit-\/credentials, etc.).
\item Validar credenciales de administrador para acceso restringido.
\item Cargar, crear y modificar contenido HTML para distintas plataformas sociales.
\item Subir, descargar, eliminar y mostrar archivos a través de la tarjeta SD.
\item Configurar la red Wi\+Fi mediante un formulario, guardando los ajustes de forma persistente.
\end{DoxyItemize}

Métodos principales\+:
\begin{DoxyItemize}
\item \doxylink{class_web_server_manager_a80f909d7a81b2bf4dbeeb71d8266c9f3}{start()}\+: Inicializa el servidor web y define sus rutas principales.
\item \doxylink{class_web_server_manager_acad523933eb6246769fa801f268dce1f}{process()}\+: Procesa las peticiones entrantes de DNS y HTTP.
\item \doxylink{class_web_server_manager_a79a4cf819803219bce7963f39a8ce613}{set\+Path()}\+: Asigna rutas de archivo o directorio en la SD para diferentes tipos (p.\+ej. índice, admin, redes sociales, etc.).
\item \doxylink{class_web_server_manager_aa50592f4b2f189d1add573e14e54ce34}{get\+HTMLContent()}\+: Lee el contenido de un archivo HTML desde la SD y reemplaza marcadores con valores dinámicos.
\item \doxylink{class_web_server_manager_ac684e61c3042eb5241caa947ea3168b8}{generate\+JSONFile()}\+: Genera un archivo JSON con la lista de archivos presentes en la tarjeta SD.
\end{DoxyItemize}

Atributos relevantes\+:
\begin{DoxyItemize}
\item m\+\_\+server, m\+\_\+dns\+Server, m\+\_\+sd\+Manager\+: Punteros para manejar el servidor web, DNS y la tarjeta SD, respectivamente.
\item m\+\_\+index\+Page, m\+\_\+exit\+Page, m\+\_\+admin\+Page, etc.\+: Variables para almacenar rutas de archivos específicos.
\item \+\_\+login\+Attempts, \+\_\+last\+Attempt\+Time\+: Controlan el número de intentos de acceso y el tiempo entre ellos.
\end{DoxyItemize}

Uso\+:
\begin{DoxyItemize}
\item Crear una instancia de \doxylink{class_web_server_manager}{Web\+Server\+Manager}, luego invocar \doxylink{class_web_server_manager_a80f909d7a81b2bf4dbeeb71d8266c9f3}{start()} para configurar y llamar \doxylink{class_web_server_manager_acad523933eb6246769fa801f268dce1f}{process()} dentro de la función {\ttfamily loop()}.
\end{DoxyItemize}

\begin{DoxyWarning}{Atención}
Es responsabilidad del usuario asegurar una correcta gestión de memoria al utilizar los punteros m\+\_\+server, m\+\_\+dns\+Server y m\+\_\+sd\+Manager
\end{DoxyWarning}
