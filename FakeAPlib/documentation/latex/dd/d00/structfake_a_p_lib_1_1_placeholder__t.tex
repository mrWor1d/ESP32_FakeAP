\doxysubsection{Descripción detallada}
Estructura que contiene marcadores de posición (placeholders) para la librería Fake\+APlib. 

Esta estructura define una colección de marcadores de posición estáticos utilizados para reemplazar valores dinámicos en plantillas HTML y otros contenidos. Los marcadores siguen el formato !\{NOMBRE\+\_\+\+MARCADOR\}.

Los marcadores incluyen\+:
\begin{DoxyItemize}
\item Información del punto de acceso Wi\+Fi (SSID, IP)
\item Estado y detalles de la conexión Wi\+Fi
\item Información de la tarjeta SD (espacio, tipo)
\item Rutas a archivos HTML de redes sociales
\end{DoxyItemize}

\begin{DoxyNote}{Nota}
Todos los marcadores son constantes String estáticas para que que se puedan llamar sin la necesidad de crear un objeto de esta clase. 
\end{DoxyNote}
