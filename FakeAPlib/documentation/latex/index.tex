Librerría personal donde se incluirán todas las dependecias y librerías externas para el desarollo de un punto de acceso falso y un servidor web utilizando la arquitectura del {\ttfamily ESP32}. Pueden mirar ejemplos de uso en la carpeta \mbox{[}examples\mbox{]}(examples).~\newline

\hypertarget{index_autotoc_md1}{}\doxysection{\texorpdfstring{Configurar el entorno}{Configurar el entorno}}\label{index_autotoc_md1}
Se tendrá que descargar esta librería y añadirla a {\ttfamily Arduino IDE} como una librería externa. Una vez hecho eso, se podrá usar en cualquier proyecto arduino en el equipo.~\newline
